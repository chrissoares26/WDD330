BOM (or the Browser Object Model

There is no standard for this model, but it is supported basically by just about every browser. The window object works with global variables and allows to check the values of those global variables using the window. syntax (ie. window.x == 6)

Example of Window objects

alert('alert Hello');
window.alert('Window alert Hello');
confirm('This is a confirmation box');
window.confirm('This is a confirmation box from the window');

The code executes the same regardless of whether it was prefixed by a window or not.

Windows opening windows

const popup = window.open ('https://spyentist.github.io/WDD330/',

HTML APIs

HTML 5.1 is the most current version of HTML and is going through faster updates in smaller increments to allow for easier upgrading.

With HTML 5 you can add data tags that are personalized towards a specific website and it's data. All you need is the data-* element in the HTML tags.

APIs are allowing for bigger data to be used with minimal transfer and quicker response times. For example session storage and local storage are used to cahce some data so that it is quicker to pull.
